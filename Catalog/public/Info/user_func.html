User architecture

Each of the devices has the following characteristics:

  • user id
  • password
  • is Super User
  • set of Buildings the user has access to
  • name
  • surname

User functionalities

Functionalities to get information of a user are available.

  • You can retrieve all users with the command:\ http://127.0.0.1:9096/users?id=all
  • You can retrieve a single user by knowing its id with the command:\ http://127.0.0.1:9096/device?id=USERID
